# Change Password Process

This document outlines the steps involved in **changing user password** using the Change Password module.

---

## 1. Open Change Password Module

Access the Change Password screen from the Web Portal.

- **Step 1.1:** Navigate to **Change Password** from the Web Portal menu.

- **Step 1.2:** The **Change Password** screen will be displayed with input fields for password modification.

  ![image1](/IMAGE/ChangePassword/image1.png)

---

## 2. Enter Old Password

Verify current password before allowing password change.

- **Step 2.1:** On the Change Password screen, enter the **Old Password** in the input field.

- **Step 2.2:** The system will validate the old password against the existing password in the database.

- **Step 2.3:** If the old password does not match, an error message will be displayed, and the user will not be able to proceed.

  ![image2](/IMAGE/ChangePassword/image2.png)

---

## 3. Enter New Password

Define a new password following security requirements.

- **Step 3.1:** Enter the **New Password** in the input field.

- **Step 3.2:** Password requirements:
  - Must be **alphanumeric** (contains both letters and numbers)
  - **Case-sensitive** (uppercase and lowercase letters are treated differently)
  - Should meet minimum length requirements as per system policy

- **Step 3.3:** Ensure the password follows the security guidelines displayed on the screen.

  ![image3](/IMAGE/ChangePassword/image3.png)

---

## 4. Confirm New Password

Re-enter the new password to ensure accuracy.

- **Step 4.1:** Enter the **Confirm Password** in the input field.

- **Step 4.2:** The system will validate that the Confirm Password matches the New Password.

- **Step 4.3:** If the passwords do not match, an error message will be displayed, and the user must re-enter the correct password.

  ![image4](/IMAGE/ChangePassword/image4.png)

---

## 5. Change Password

Save the new password to the system.

- **Step 5.1:** After successfully entering and matching all password fields, click the **Change Password** button.

- **Step 5.2:** The system will update the password in the database.

- **Step 5.3:** A confirmation message will appear upon successful password change.

- **Step 5.4:** The user will be required to log in again with the new password.

  ![image5](/IMAGE/ChangePassword/image5.png)

---

## Summary

- **Open Change Password:** Access the Change Password module from the Web Portal menu.
- **Old Password Validation:** Enter old password which must match the existing password in the system.
- **New Password Entry:** Enter new alphanumeric, case-sensitive password following security requirements.
- **Confirm Password Match:** Re-enter new password to ensure both entries match.
- **Change Password:** Click Change Password button to save and update the password successfully.
