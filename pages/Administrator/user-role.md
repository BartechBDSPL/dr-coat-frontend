# User Role – Creation, Display & Update Process

This document outlines the steps involved in **creating a user role**, **viewing role details**, and **updating user role information** using the User Role module.

---

## 1. Open User Role Module

Access the User Role management screen from the Web Portal.

- **Step 1.1:** Navigate to **User Role** from the Web Portal menu.

- **Step 1.2:** The **User Role** screen will be displayed showing existing roles (if any).
  ![image1](/IMAGE/UserRole/image1.png)

---

## 2. Create New User Role

Define a new user role by entering role name and setting access permissions.

- **Step 2.1:** On the User Role screen, enter the **Role Name** in the input field.

- **Step 2.2:** Select access type:
  - **Web Access** – Enable this checkbox if the role should have access to the Web Portal
  - **HHT Access** – Enable this checkbox if the role should have access to the Handheld Terminal (HHT)

- **Step 2.3:** Configure additional permissions or settings as required.
  ![image2](/IMAGE/UserRole/IMAGE2.png)
  ![image3](/IMAGE/UserRole/IMAGE3.png)
---

## 3. Save User Role

Save the newly created user role to the system.

- **Step 3.1:** Click the **Save** button to create the user role.

- **Step 3.2:** A confirmation message will appear upon successful creation.
 
  ![image4](/IMAGE/UserRole/IMAGE4.png)

---

## 4. Edit User Role

Update existing user role details including role name and access permissions.

- **Step 4.1:** Select the user role record from the list.

- **Step 4.2:** Click the **Edit** button to enable editing mode.
  ![image5](/IMAGE/UserRole/IMAGE5.png)

- **Step 4.3:** All role details will be displayed in the input fields:
  - Role Name
  - Web Access checkbox
  - HHT Access checkbox
  - Other configurable permissions

- **Step 4.4:** Modify the required fields as needed.

- **Step 4.5:** Click the **Update** button to save the changes.
  ![image6](/IMAGE/UserRole/IMAGE6.png)


---

## Summary

- **Open User Role:** Access the User Role module to view existing roles.
- **Create Role:** Enter role name, select Web and HHT access, then save.
- **Edit Role:** Click Edit to display all details in input fields, modify, and update.
