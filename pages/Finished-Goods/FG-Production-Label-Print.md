# FG Production Order Label Printing Process

This document outlines the steps involved in **printing labels for Finished Goods (FG) Production Orders**, including production order selection, detail retrieval, serial number generation, and label printing.

---

## 1. Open FG Production Order Label Printing Module

Access the FG Production Order Label Printing screen from the Web Portal.

- **Step 1.1:** Navigate to **FG Production Order Label Printing** from the Web Portal menu.

- **Step 1.2:** The **FG Production Order Label Printing** screen will be displayed with input fields and options.

  <img src="/IMAGE/FGTransactions/FGProductionLabelPrinting/image9.png" width="400" alt="image1">

---

## 2. Select Production Order Number

Choose or enter the production order for which labels need to be printed.

- **Step 2.1:** Select the **Production Order Number** from the dropdown list, or manually enter the production order number in the input field.

- **Step 2.2:** Ensure the correct production order is selected before proceeding.

  <img src="/IMAGE/FGTransactions/FGProductionLabelPrinting/image1.png" width="400" alt="image2">

---

## 3. Get Production Order Details

Retrieve the details associated with the selected production order.

- **Step 3.1:** After selecting or entering the production order number, click the **Get Details** button.

- **Step 3.2:** The system will fetch and display the production order details including product information, quantities, and other relevant data.

  <img src="/IMAGE/FGTransactions/FGProductionLabelPrinting/image3.png" width="400" alt="image3">

---

## 4. Generate Serial Numbers

Enter quantity and weight information to generate serial numbers for label printing.

- **Step 4.1:** In the **Label Generation** section, enter the following information:
  - **Total Quantity (Kgs)** – This field is **editable**
    **Note:- Suppose the total quantity of Production Order is 1000 and user want to print the label of 500 quantity only then he will edit the total quantity field to 500.**
  - **Base Weight (Kgs)** – Enter the base weight in kilograms  
  - **Tare Weight (Kgs)** – Enter the tare weight in kilograms  

- **Step 4.2:** The **Total Weight (Kgs)** will be calculated automatically.

- **Step 4.3:** Click the **Generate Serial Numbers** button to create serial numbers based on the entered quantity and weight information.

  <img src="/IMAGE/FGTransactions/FGProductionLabelPrinting/image4.png" width="400" alt="image4">
---

## 5. View Generated Serial Numbers

Review the generated serial numbers along with date and printer information.

- **Step 5.1:** After clicking Generate Serial Numbers, the system will display:
  - **Date** – Current date or production date
  - **Printer Information** – Available printer(s) for label printing
  - **Serial Numbers List** – All generated serial numbers displayed below

- **Step 5.2:** Review all serial numbers to ensure accuracy before printing.

  <img src="/IMAGE/FGTransactions/FGProductionLabelPrinting/image5.png" width="400" alt="image5">

---

## 6. Print Labels

Select printer settings and print the generated labels.

- **Step 6.1:** Select or verify the **Date** field if modification is required.

- **Step 6.2:** Select the **Printer** from the printer information dropdown list.

- **Step 6.3:** Ensure all serial numbers to be printed are displayed and correct.

- **Step 6.4:** Click the **Print Labels** button to send the labels to the selected printer.

- **Step 6.5:** A confirmation message will appear upon successful printing, and labels will be printed with the serial numbers.

  <img src="/IMAGE/FGTransactions/FGProductionLabelPrinting/image6.png" width="400" alt="image6">

---

## Summary

- **Open Module:** Access the FG Production Order Label Printing module from the Web Portal menu.
- **Select Production Order:** Choose production order from dropdown or enter manually.
- **Get Details:** Click Get Details to retrieve production order information.
- **Generate Serial Numbers:** Enter Base Weight and Tare Weight, then click Generate Serial Numbers.
- **View Information:** Review date, printer information, and all generated serial numbers.
- **Print Labels:** Select date and printer, then click Print Labels to complete the printing process.
