# Shipment Return Module Process

This document outlines the steps involved in **accessing the Shipment Return module**, **scanning material barcodes**, **viewing material details**, **selecting return reasons**, and **completing return operations** using the Shipment Return Module.

---

## 1. Open Shipment Return Module

Access the Shipment Return screen from the application menu.

- **Step 1.1:** Navigate to **Shipment Return** from the application menu.

- **Step 1.2:** The **Shipment Return** screen will be displayed with material barcode scanning and return processing options.

  <img src="/IMAGE/FGShipmentReturn/image1.png" width="400" alt="image1">

---

## 2. Scan Material Barcode

Scan the material barcode to retrieve details for the return operation.

- **Step 2.1:** Focus on the **Material Barcode** scan field on the Shipment Return screen.

- **Step 2.2:** Scan the material barcode using the scanner device.

- **Step 2.3:** The system will validate the scanned barcode.

### 2.1 Invalid Barcode Scenario

- **Step 2.3.1:** If the scanned barcode is invalid or not found in the system, an error message will be displayed.

- **Step 2.3.2:** The user must scan a valid material barcode to proceed with the return operation.

  <img src="/IMAGE/FGShipmentReturn/image3.png" width="400" alt="image2">

### 2.2 Valid Barcode Scenario

- **Step 2.3.3:** If the scanned barcode is valid, the system will retrieve and display all material details.

- **Step 2.3.4:** Material information including Material Code, Material Description, Quantity, Batch No, Shipment Number, Customer Details, and other relevant information will be displayed on the screen.

- **Step 2.3.5:** The material details will be ready for the return process.

  <img src="/IMAGE/FGShipmentReturn/image2.png" width="400" alt="image3">
  

---

## 3. Initiate Return Material Process

Begin the material return operation for the scanned material.

- **Step 3.1:** After scanning the valid barcode and reviewing the displayed material details, verify that all information is correct.

- **Step 3.2:** Locate the **Return Material** button on the screen.

- **Step 3.3:** Click the **Return Material** button to initiate the return process.

- **Step 3.4:** The system will display a dialog box prompting for the return reason.

  <img src="/IMAGE/FGShipmentReturn/image4.png" width="400" alt="image4">
 

---

## 4. Select Return Reason

Provide a mandatory return reason for the material being returned.

- **Step 4.1:** After clicking the Return Material button, a **Return Reason** dialog box will be displayed.

- **Step 4.2:** The **Return Reason** field is mandatory and must be filled before proceeding with the return operation.

  <img src="/IMAGE/FGShipmentReturn/image5.png" width="400" alt="image5">


---

## 5. Confirm Return and Complete Operation

Confirm the return operation and receive success confirmation.

- **Step 5.1:** After selecting the return reason, verify that all details including material information and return reason are correct.

- **Step 5.2:** Click the **OK** button to confirm and complete the return operation.

- **Step 5.3:** The system will process the return transaction and update the inventory records.

- **Step 5.4:** A success message will be displayed confirming that the material has been successfully returned.

- **Step 5.5:** The returned material will be recorded in the system with the selected return reason.

- **Step 5.6:** The inventory will be updated to reflect the returned material status and location.

- **Step 5.7:** The return operation is now complete, and the material is available for further processing such as inspection, restocking, or disposal based on the return reason.

  <img src="/IMAGE/FGShipmentReturn/image6.png" width="400" alt="image6">
---

## Summary

- **Open Shipment Return Module:** Click on Shipment Return menu to display the Shipment Return screen.
- **Scan Material:** Scan material barcode to retrieve details. System displays error message for invalid barcodes or shows complete material information for valid barcodes.
- **Initiate Return:** After scanning valid barcode and reviewing details, click Return Material button to begin the return process and display the return reason dialog.
- **Select Return Reason:** Return Reason is mandatory. Select the appropriate return reason from the dropdown and verify all details.
- **Confirm Return:** Click OK button to complete the return operation. System processes the transaction and displays success message confirming the material has been successfully returned.
