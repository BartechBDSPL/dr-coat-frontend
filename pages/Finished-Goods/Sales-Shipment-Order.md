# Sales Shipment Order Module Process

This document outlines the steps involved in **accessing the Sales Shipment Order module**, **filtering shipment orders**, **retrieving order details**, **selecting warehouse**, **assigning orders to users**, and **viewing recent assignments** using the Sales Shipment Order Module.

---

## 1. Open Sales Shipment Order Module

Access the Sales Shipment Order screen from the application menu.

- **Step 1.1:** Navigate to **Sales Shipment Order** from the application menu.

- **Step 1.2:** The **Sales Shipment Order** screen will be displayed with date filter, shipment selection, and order assignment options.

  <img src="/IMAGE/SalesShipmentOrder/image1.png" width="400" alt="image1">

---

## 2. Apply Date Filter and Fetch Shipments

Filter and retrieve shipment orders based on selected date range.

- **Step 2.1:** Locate the **Date Filter** section on the Sales Shipment Order screen.

- **Step 2.2:** Select the desired date or date range from the date picker to filter shipment orders.

- **Step 2.3:** Click the **Fetch Shipments** button to retrieve shipment orders for the selected date range.

- **Step 2.4:** The system will fetch all available shipment orders matching the date criteria.

- **Step 2.5:** The fetched shipment orders will be available for selection in the Shipment Number dropdown or input field.

  <img src="/IMAGE/SalesShipmentOrder/image2.png" width="400" alt="image2">
 
---

## 3. Select Shipment Number and Get Details

Choose a specific shipment order and retrieve its complete details.

- **Step 3.1:** Locate the **Shipment Number** dropdown or input field.

- **Step 3.2:** Either select the desired Shipment Number from the dropdown list or manually enter the shipment number.

- **Step 3.3:** Click the **Get Details** button to retrieve the complete information for the selected shipment order.

- **Step 3.4:** The system will fetch and display all relevant shipment order details on the screen.

- **Step 3.5:** Order details including customer information, item details, quantities, and shipment information will be displayed.

  <img src="/IMAGE/SalesShipmentOrder/image3.png" width="400" alt="image3">



---

## 4. Select Warehouse Code

Choose the warehouse associated with the shipment order for user assignment.

- **Step 4.1:** After retrieving the shipment order details, locate the **Warehouse Code** dropdown field.

- **Step 4.2:** Click on the **Warehouse Code** dropdown to view all available warehouses.

- **Step 4.3:** Select the appropriate Warehouse Code from the dropdown list.

- **Step 4.4:** After selecting the Warehouse Code, the system will enable the **Assign to User** dropdown field.

- **Step 4.5:** The system will automatically load all users assigned to the selected warehouse.

  <img src="/IMAGE/SalesShipmentOrder/image4.png" width="400" alt="image4">  

## 5. Select User for Assignment

Choose the user who will be assigned to process the shipment order.

- **Step 5.1:** After the Warehouse Code is selected and the **Assign to User** dropdown is enabled, click on the dropdown.

- **Step 5.2:** The system will display a list of all users associated with the selected warehouse.

- **Step 5.3:** Review the list of available users.

- **Step 5.4:** Select the desired user from the dropdown list who will be responsible for processing the shipment order.

- **Step 5.5:** Verify that the correct user has been selected before proceeding with the assignment.

  <img src="/IMAGE/SalesShipmentOrder/image5.png" width="400" alt="image5">
  
---

## 6. Assign Shipment Order to User

Complete the assignment process by assigning the shipment order to the selected user.

- **Step 6.1:** After selecting the user, verify all details including Shipment Number, Warehouse Code, and User are correct.

- **Step 6.2:** Click the **Assign** button to assign the shipment order to the selected user.

- **Step 6.3:** The system will process the assignment and update the shipment order status.

- **Step 6.4:** The material and shipment order will be assigned to the selected user.

- **Step 6.5:** A confirmation message will be displayed confirming the successful assignment.

- **Step 6.6:** The assigned user will be notified and can proceed with processing the shipment order.
  <img src="/IMAGE/SalesShipmentOrder/image6.png" width="400" alt="image6">
  


---

## 7. View Recent Shipment Orders

Review the list of recently assigned shipment orders.

- **Step 7.1:** After successfully assigning the shipment order, the system will display the **Recent Shipment Orders** section.

- **Step 7.2:** The Recent Shipment Orders list will show all recently assigned or processed shipment orders.

- **Step 7.3:** Review the recent shipment orders to verify that the assignment was completed successfully.

- **Step 7.4:** The recent orders list provides a quick reference for tracking shipment assignments and their current status.

- **Step 7.5:** You can use this list to monitor ongoing shipments and ensure timely processing.

 <img src="/IMAGE/SalesShipmentOrder/image7.png" width="400" alt="image7">

---

## Summary

- **Open Sales Shipment Order Module:** Click on Sales Shipment Order menu to display the Sales Shipment Order screen.
- **Apply Date Filter:** Select date or date range from date picker and click Fetch Shipments to retrieve shipment orders for the selected period.
- **Get Shipment Details:** Select or enter Shipment Number and click Get Details to retrieve complete shipment order information.
- **Select Warehouse:** Click on Warehouse Code dropdown to select the appropriate warehouse, which enables the Assign to User dropdown.
- **Select User:** Click on Assign to User dropdown to display all users associated with the selected warehouse and select the desired user.
- **Assign Order:** Click Assign button to assign the shipment order to the selected user and display confirmation message.
- **View Recent Orders:** System displays Recent Shipment Orders section showing all recently assigned or processed shipment orders with their details.
