# User Master – User Creation, First Login & Update Process

This document outlines the steps involved in **creating a user**, **first-time login with mandatory password change**, and **updating user details** using the User Master module.

---

## 1. Create New User (User Master)

When an administrator creates a new user, all mandatory details must be filled before saving.

- **Step 1.1:** Navigate to **User Master** from the Web Portal.

- **Step 1.2:** Fill all required fields in the User Master form:
   
  - Any other mandatory fields
  <img src="/IMAGE/UserMaster/image1.png" width="400" alt="image1">
  

- **Step 1.3:** Click the **Save** button to create the user.
  <img src="/IMAGE/UserMaster/IMAGE2.png" width="400" alt="image2">
  <img src="/IMAGE/UserMaster/IMAGE3.png" width="400" alt="image3">

---

## 2. First-Time Login & Change Password

This process is triggered when the user logs in for the first time.

- **Step 2.1:** Enter **Username** and **Password** on the login screen.

- **Step 2.2:** Click the **Login** button.

- **Step 2.3:** On first login, a **Change Password** popup will appear automatically.
  <img src="/IMAGE/UserMaster/IMAGE4.png" width="400" alt="image4">

- **Step 2.4:** Enter:
  - **Current Password**
  - **New Password**
  - **Confirm New Password**

- **Step 2.5:** Click the **Change Password** button.
  - After successful password change, the system will **log in automatically**.
  <img src="/IMAGE/UserMaster/IMAGE5.png" width="400" alt="image5">
  <img src="/IMAGE/UserMaster/IMAGE6.png" width="400" alt="image6">
---

## 3. Update User Details (Edit User)

This section is used to modify existing user information.

- **Step 3.1:** Go to **User Master** on the Web Portal.

- **Step 3.2:** Select the user record from the list.

- **Step 3.3:** Click the **Edit** button to enable editing.
  <img src="/IMAGE/UserMaster/IMAGE7.png" width="400" alt="image7">

- **Step 3.4:** Update required fields such as:
  - Warehouse Code  
  - User Role  
  - Contact details  
  - Other editable fields

- **Step 3.5:** Click the **Update** button to save the changes.
  <img src="/IMAGE/UserMaster/IMAGE8.png" width="400" alt="image8">

---

## Summary

- **User Creation:** Admin fills all mandatory fields and saves the user.
- **First Login:** User must change password; system auto-logs in after success.
- **User Update:** Admin can edit and update user details anytime through User Master.
