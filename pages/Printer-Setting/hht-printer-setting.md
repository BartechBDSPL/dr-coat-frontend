# Printer Setting Module Process

This document outlines the steps involved in **accessing the Printer Setting module**, **selecting module and printer**, **saving printer configuration**, and **viewing saved settings** using the Printer Setting Module.

---

## 1. Open Printer Setting Module

Access the Printer Setting screen from the application menu.

- **Step 1.1:** Navigate to **Printer Setting** from the application menu.

- **Step 1.2:** The **Printer Setting** screen will be displayed with module selection, printer selection, and configuration options.

  ![image1](/IMAGE/HHTPrinterSetting/image1.png)

---

## 2. Select Module Name

Choose the module for which the printer configuration needs to be set.

- **Step 2.1:** Locate the **Module Name** dropdown field on the Printer Setting screen.

- **Step 2.2:** Click on the **Module Name** dropdown to view all available modules in the system.

- **Step 2.3:** The system will display a list of modules such as Material Split, FG Stock Transfer, Put Away, Internal Movement, and other available modules.

- **Step 2.4:** Select the desired module from the dropdown list for which you want to configure the printer settings.

 ![image1](/IMAGE/HHTPrinterSetting/image1.png)
  

---

## 3. Select Printer Name

Choose the printer that will be associated with the selected module.

- **Step 3.1:** After selecting the Module Name, locate the **Printer Name** dropdown field.

- **Step 3.2:** Click on the **Printer Name** dropdown to view all available printers connected to the system.

- **Step 3.3:** The system will display a list of all configured and available printers.

- **Step 3.4:** Review the list of printers and select the appropriate printer that should be used for the selected module.

- **Step 3.5:** The selected printer will be associated with the module for all printing operations.

  ![image2](/IMAGE/HHTPrinterSetting/image2.png)

 

---

## 4. Save Printer Configuration

Save the printer settings and view the configuration in the settings table.

- **Step 4.1:** After selecting both the Module Name and Printer Name, verify that all selections are correct.

- **Step 4.2:** Click the **Save Printer** button to save the printer configuration for the selected module.

- **Step 4.3:** The system will process the configuration and save the printer settings.

- **Step 4.4:** A success message will be displayed confirming that the printer settings have been saved successfully.

- **Step 4.5:** The saved printer configuration details will be automatically displayed in the table below the configuration section.

- **Step 4.6:** The table will show all saved printer settings including Module Name, Printer Name relevant details.

- **Step 4.7:** You can review the table to verify that the printer configuration has been correctly saved and associated with the module.

  ![image3](/IMAGE/HHTPrinterSetting/image3.png)

---

## Summary

- **Open Printer Setting Module:** Click on Printer Setting menu to display the Printer Setting screen.
- **Select Module:** Click on Module Name dropdown to display all available modules and select the desired module for printer configuration.
- **Select Printer:** Click on Printer Name dropdown to display all available printers and select the appropriate printer.
- **Save Configuration:** Click Save Printer button to save the printer settings. System displays success message and shows saved configuration details in the table below.
