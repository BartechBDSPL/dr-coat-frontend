# Printer Master – Creation, Display & Update Process

This document outlines the steps involved in **creating a printer**, **viewing printer details**, and **updating printer information** using the Printer Master module.

---

## 1. Open Printer Master Module

Access the Printer Master management screen from the Web Portal.

- **Step 1.1:** Navigate to **Printer Master** from the Web Portal menu.

- **Step 1.2:** The **Printer Master** screen will be displayed showing existing printers (if any).
  ![image1](/IMAGE/PrinterMaster/image1.png)

---

## 2. Create New Printer

Define a new printer by entering required information.

- **Step 2.1:** On the Printer Master screen, fill in all the **required fields**

- **Step 2.2:** Ensure all mandatory fields are filled with accurate information.
  ![image2](/IMAGE/PrinterMaster/image2.png)

---

## 3. Save Printer

Save the newly created printer to the system.

- **Step 3.1:** Click the **Save** button to create the printer record.

- **Step 3.2:** A confirmation message will appear upon successful creation.

- **Step 3.3:** The new printer will be added to the printer list.
  ![image3](/IMAGE/PrinterMaster/image3.png)

---

## 4. Edit Printer

Update existing printer details including configuration and settings.

- **Step 4.1:** Select the printer record from the list that you want to edit.

- **Step 4.2:** Click the **Edit** button to enable editing mode.
  **Step 4.3:** All printer details will be displayed in the input field

- **Step 4.4:** Modify the required fields as needed.
  ![image4](/IMAGE/PrinterMaster/image4.png)


---

## 5. Update Printer

Save the changes made to the printer information.

- **Step 5.1:** After modifying the required fields, click the **Update** button to save the changes.

- **Step 5.2:** A confirmation message will appear upon successful update.

- **Step 5.3:** The updated printer information will be reflected in the printer list.
  ![image5](/IMAGE/PrinterMaster/image5.png)

---

## Summary

- **Open Printer Master:** Access the Printer Master module to view existing printers.
- **Create Printer:** Fill in all required fields including printer name, IP address, port, type, location, and status, then click Save.
- **Edit Printer:** Select printer from list and click Edit to display all details in input fields.
- **Update Printer:** Modify required fields and click Update to save changes.
