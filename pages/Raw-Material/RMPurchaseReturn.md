# Purchase Return Process

This document outlines the steps involved in scanning material, verifying details, and completing the purchase return process.

---

## 1. Scan Material

The user needs to scan the material barcode to initiate the return process.

- **Step 1.1:** Navigate to the **Purchase Return** module.

- **Step 1.2:** The **Purchase Return** screen will be displayed.

- **Step 1.3:** Scan the material barcode using the scanner.

  <img src="/IMAGE_RM/RMPurchaseReturn/IMAGE1.png" width="400" alt="image1">

---

## 2. Validate Scanned Material

After scanning, the system validates the barcode and displays details.

- **Step 2.1:** A message **"Valid barcode scanned"** will be displayed.

- **Step 2.2:** The system will show the following details:
  - GRN No  
  - Item Code  
  - Item Description  
  - Quantity  
  - Lot Number  
  - Vendor Code & Name  
  - Serial Number  

  <img src="/IMAGE_RM/RMPurchaseReturn/IMAGE2.png" width="400" alt="image2">

---

## 3. Remove Scanned Barcode (Optional)

If needed, the scanned barcode can be removed.

- **Step 3.1:** Click on the **Remove** checkbox.

- **Step 3.2:** The scanned data will be cleared.

- **Step 3.3:** Scan the barcode again if required.

  <img src="/IMAGE_RM/RMPurchaseReturn/IMAGE3.png" width="400" alt="image3">

---

## 4. Initiate Return

Proceed to return the scanned material.

- **Step 4.1:** Click on the **Return** button.

- **Step 4.2:** A popup will appear asking for the **Reason for Return**.

- **Step 4.3:** Enter the reason (mandatory field).

- **Step 4.4:** Click on **OK** to confirm.

  <img src="/IMAGE_RM/RMPurchaseReturn/IMAGE4.png" width="400" alt="image4">

  <img src="/IMAGE_RM/RMPurchaseReturn/IMAGE5.png" width="400" alt="image5">

---

## 5. Return Confirmation

After successful submission, the system confirms the return.

- **Step 5.1:** A success message will be displayed:  
  **"Purchase return done successfully."**

  
<img src="/IMAGE_RM/RMPurchaseReturn/IMAGE6.png" width="400" alt="image6">
---

**Note :- After clicking on return button the item or material will no longer can be scanned in any other module as this module is used to return the material in case of any issue. Only materials with Bartech barcode will be used in this module.**
