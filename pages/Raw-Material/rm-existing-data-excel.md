# RM Existing Data Upload Process

This document outlines the steps involved in uploading existing RM data using Excel, generating labels, and printing them.

---

## 1. Upload RM Existing Data Excel

The user needs to upload an Excel file containing RM data.

- **Step 1.1:** Navigate to the **RM Existing Data Upload** screen.

- **Step 1.2:** Click on **Choose File** and select the Excel file.

- **Step 1.3:** Click on the **Upload** button.

- **Step 1.4:** You can download the **Sample Format** for reference.
  - Ensure that the **Excel header matches exactly** with the sample file.

  <img src="/IMAGE_RM/RMExistingDataExcel/IMAGE1.png" width="400" alt="image1">

---

## 2. Select Item Code and Lot Number

After uploading the Excel file, select item details.

- **Step 2.1:** Select the **Item Code** from the dropdown.

- **Step 2.2:** Based on the selected item code, the **Lot Number** dropdown will display values from the uploaded Excel.

- **Step 2.3:** Select the **Lot Number**.

- **Step 2.4:** The system will display details related to:
  - Item Code  
  - Item Description  
  - Lot Number  
  - Total Quantity  
  - Printed Quantity  
  - Remaining Quantity  

  <img src="/IMAGE_RM/RMExistingDataExcel/IMAGE2.png" width="400" alt="image2">

---

## 3. Label Generation

Enter required details to generate labels.

- **Step 3.1:** In the **Label Generation** section, enter:
  - **Total Quantity** if required
    **Note:- Suppose the total quantity of Production Order is 1000 and user want to print the label of 500 quantity only then he will edit the total quantity field to 500.**

  - **Base Weight (KGS)**  
  - **Tare Weight (KGS)**  

- **Step 3.2:** The **Total Weight (KGS)** will be **calculated automatically**.

- **Step 3.3:** Click on **Generate Serial Numbers**.

  <img src="/IMAGE_RM/RMExistingDataExcel/IMAGE3.png" width="400" alt="image3">

---

## 4. Enter Date & Printer Details

Provide manufacturing and printing details.

- **Step 4.1:** Select the **Manufacturing Date**.

- **Step 4.2:** The **Expiry Date** will be **auto-calculated (1 year increment)**.

- **Step 4.3:** Select the **Printer** from the dropdown.

  <img src="/IMAGE_RM/RMExistingDataExcel/IMAGE4.png" width="400" alt="image4">

---

## 5. Confirm and Print Labels

Verify details before printing.

- **Step 5.1:** Click on the **Print Labels** button.

- **Step 5.2:** A confirmation popup will appear.

- **Step 5.3:** Verify the following details:
  - Base Weight  
  - Tare Weight  
  - Total Weight  
  - Serial Numbers  
  - Item Code & Lot Number  
  - Manufacturing & Expiry Date  
  - Printer  

- **Step 5.4:** Click on the **Print** button to proceed.

  <img src="/IMAGE_RM/RMExistingDataExcel/IMAGE5.png" width="400" alt="image5">

---

## 6. Success Message

- After successful printing, the system will display a confirmation message.
