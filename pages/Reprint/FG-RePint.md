# Reprint Label Request & Approval Process

This document outlines the steps involved in **requesting label reprint**, **approving reprint requests**, and **printing approved labels** using the Reprint Request and Reprint Approval modules.

---

## 1. Open Reprint Request Module

Access the Reprint Request screen from the Web Portal.

- **Step 1.1:** Navigate to **Reprint Request** from the Web Portal menu.

- **Step 1.2:** The **Reprint Request** screen will be displayed with search and filter options.

  ![image1](/IMAGE/RePrint/Reprint-Request/image1.png)

---

## 2. Search for Printed Labels

Retrieve the list of previously printed labels available for reprint.

- **Step 2.1:** On the Reprint Request screen, click the **Search** button.

- **Step 2.2:** The system will display all printed labels with details including serial numbers, production order, date, and other relevant information.

  ![image2](/IMAGE/RePrint/Reprint-Request/image2.png)

---

## 3. Request Reprint

Select labels for reprint and submit the request.

- **Step 3.1:** Select the **checkbox** next to the label(s) you want to reprint from the displayed list.

- **Step 3.2:** Click the **Request Reprint** button to initiate the reprint request.

  ![image3](/IMAGE/RePrint/Reprint-Request/image3.png)
  ![image4](/IMAGE/RePrint/Reprint-Request/image4.png)

---

## 4. Enter Reprint Reason

Provide a mandatory reason for the reprint request.

- **Step 4.1:** A **Reason Popup Box** will appear on the screen.

- **Step 4.2:** Enter the **Reason** in the input field. This field is **mandatory** and cannot be left blank.

- **Step 4.3:** Click **Submit** or **OK** to confirm and send the reprint request for approval.

- **Step 4.4:** The request will be sent to the **Reprint Approval Module** for administrator review.

  ![image4](/IMAGE/RePrint/Reprint-Request/image5.png)
  ![image4](/IMAGE/RePrint/Reprint-Request/image6.png)
---

## 5. Access Reprint Approval Module

Administrator reviews and approves or rejects reprint requests.

- **Step 5.1:** Navigate to **Administrator Reprint Approval Module** from the Web Portal menu.

- **Step 5.2:** The **Reprint Approval** screen will be displayed showing all pending reprint requests.

  ![image5](/IMAGE/RePrint/Reprint-Approval/image1.png)

---

## 6. Review Reprint Request Details

View complete information about the reprint request before approval.

- **Step 6.1:** Select the reprint request from the list by clicking on it.

- **Step 6.2:** All request details will be displayed.

- **Step 6.3:** Review all information carefully before proceeding.

  ![image6](/IMAGE/RePrint/Reprint-Approval/image2.png)

---

## 7. Approve Reprint Request

Approve the reprint request to enable label printing.

- **Step 7.1:** After reviewing the request details, **swipe to approve** the reprint request.

- **Step 7.2:** The system will update the request status to **Approved**.

- **Step 7.3:** The approved request will be sent back to the **Reprint Request Module** with a **Print Button** enabled.

  ![image7](/IMAGE/RePrint/Reprint-Approval/image3.png)

---

## 8. Print Approved Labels

Print the approved labels from the Reprint Request module.

- **Step 8.1:** Return to the **Reprint Request Module** screen.

- **Step 8.2:** The approved reprint request will now display a **Print Button**.

- **Step 8.3:** Click the **Print** button.

  ![image8](/IMAGE/RePrint/Reprint-Request/image8.png)

---

## 9. Select Printer and Print

Choose printer and complete the printing process.

- **Step 9.1:** A **Select Printer Popup** will appear on the screen.

- **Step 9.2:** Select the desired **Printer** from the dropdown list.

- **Step 9.3:** Click the **Print** button in the popup to send the labels to the selected printer.

- **Step 9.4:** A confirmation message will appear upon successful printing.

  ![image9](/IMAGE/RePrint/Reprint-Request/image9.png)

---

## 10. Reject Reprint Request (Optional)

Administrator can reject reprint requests if necessary.

- **Step 10.1:** In the **Reprint Approval Module**, select the reprint request to review.

- **Step 10.2:** **Swipe to reject** the reprint request if it does not meet approval criteria.

- **Step 10.3:** The reprint label will be **removed from the Reprint Approval Module**.

- **Step 10.4:** The request status will be updated to **Rejected**, and the requester will be notified.

  ![image10](/IMAGE/RePrint/Reprint-Approval/image4.png)

---

## Summary

- **Open Reprint Request Module:** Access the module and click Search to display printed labels.
- **Request Reprint:** Select checkbox for desired labels and click Request Reprint.
- **Enter Reason:** Provide mandatory reason in the popup box and submit.
- **Approval Process:** Administrator accesses Reprint Approval Module to review requests.
- **Review Details:** Select request to display data in fields above.
- **Approve Request:** Swipe to approve, enabling Print Button in Reprint Request module.
- **Print Labels:** Click Print, select printer from popup, and complete printing.
- **Reject Request:** Administrator can swipe to reject, removing the request from approval list.
