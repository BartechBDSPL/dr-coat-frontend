# Stock Transfer Order Assignment Process

This document outlines the steps involved in **fetching stock transfer orders**, **retrieving order details**, **selecting warehouse**, and **assigning transfer orders to users** using the Stock Transfer Order module.

---

## 1. Open Stock Transfer Order Module

Access the Stock Transfer Order screen from the Web Portal.

- **Step 1.1:** Navigate to **Stock Transfer Order** from the Web Portal menu.

- **Step 1.2:** The **Stock Transfer Order** screen will be displayed with date filter and order management options.
  ![image1](/IMAGE/StockTransferOrder/image1.png)
  

---

## 2. Fetch Transfer Orders

Filter and retrieve transfer orders based on selected date range.

- **Step 2.1:** Select the **Date Filter** by choosing the desired date or date range from the date picker.

- **Step 2.2:** Click the **Fetch Order** button to retrieve transfer orders for the selected date range.

- **Step 2.3:** The system will fetch all available transfer orders and display them in the **Transfer Order No** dropdown list.
  ![image2](/IMAGE/StockTransferOrder/image2.png)
  ![image3](/IMAGE/StockTransferOrder/image3.png)

---

## 3. Get Transfer Order Details

Select a specific transfer order and retrieve its complete details.

- **Step 3.1:** Select the desired **Transfer Order No** from the dropdown list.

- **Step 3.2:** Click the **Get Details** button to retrieve the complete information for the selected transfer order.

- **Step 3.3:** The system will fetch and display all relevant transfer order details on the screen.

  ![image4](/IMAGE/StockTransferOrder/image4.png)

---

## 4. Select Warehouse Code

Choose the warehouse associated with the transfer order.

- **Step 4.1:** After getting the order details, the **Warehouse Code** field will be displayed.

- **Step 4.2:** Select the appropriate **Warehouse Code** from the dropdown list.

- **Step 4.3:** The warehouse selection is required to proceed with user assignment.
  ![image5](/IMAGE/StockTransferOrder/image5.png)
  ![image6](/IMAGE/StockTransferOrder/image6.png)

---

## 5. Load Users for Selected Warehouse

View all users associated with the selected warehouse.

- **Step 5.1:** After selecting the Warehouse Code, the system will automatically load all users assigned to that warehouse.

- **Step 5.2:** The **User** dropdown list will be populated with all available users for the selected warehouse.

- **Step 5.3:** Review the list of users to select the appropriate user for transfer order assignment.
  ![image7](/IMAGE/StockTransferOrder/image7.png)

---

## 6. Assign Transfer Order to User

Select a user and assign the transfer order for execution.

- **Step 6.1:** Select the desired **User** from the dropdown list.

- **Step 6.2:** Verify all details including Transfer Order No, Warehouse Code, and User are correct.

- **Step 6.3:** Click the **Assign Transfer Order** button to assign the transfer order to the selected user.

- **Step 6.4:** A confirmation message will appear upon successful assignment.

- **Step 6.5:** The assigned user will be notified and can proceed with executing the transfer order.
  ![image8](/IMAGE/StockTransferOrder/image8.png)
  ![image9](/IMAGE/StockTransferOrder/image9.png)

---

## Summary

- **Open Stock Transfer Order Module:** Access the module to manage transfer orders.
- **Fetch Orders:** Select date filter and click Fetch Order to display available transfer orders in dropdown.
- **Get Details:** Select Transfer Order No from dropdown and click Get Details.
- **Select Warehouse:** Choose the appropriate Warehouse Code from the displayed options.
- **Load Users:** System automatically loads all users assigned to the selected warehouse.
- **Assign Order:** Select user and click Assign Transfer Order to complete the assignment process.
