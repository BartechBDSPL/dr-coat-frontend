# Material Inward Process Documentation

This document provides a detailed step-by-step guide for performing the Material Inward process. It includes instructions for scanning material codes, handling scanned data, removing scanned items, and completing the inward process.

---

## 1. Accessing the Material Inward Page

 ### **Step 1.1:** After successful login, the **FG Material Movement Menu** opens.

 ### **Step 1.2:** In the menu, click on the **Material Inward** option.

  <img src="/DOCS_IMAGE/hht-inward/image0.jpeg" alt="Material Inward Option" width="250px"/>

 ### **Step 1.3:** The **Material Inward Page** will open.
  

---

## 2. Scanning Material Codes

### 2.1 Scan Material Code Field
- **Step 2.1.1:** Locate the field labeled **Scan Material Code** on the Material Inward Page.
- **Step 2.1.2:** Use the scanner to scan the **Box Number** in the designated field.

  <img src="/DOCS_IMAGE/hht-inward/image1.jpeg" alt="Material Inward Page" width="250px"/>

### 2.2 Quality Check Verification
- **Step 2.2.1:** If the **Quality Check (QC)** status is pending for the scanned box or pallet number:
  - The system prevents the inward process.
  - An error message is displayed in red: **"QC is pending or Not Yet Done!."**

    <img src="/DOCS_IMAGE/hht-inward/image2.jpeg" alt="QC Pending Message" width="250px"/>

### 2.3 Valid Scans
- **Step 2.3.1:** If the scanned Box Number is correct:
  - A green message appears: **"Barcode Scanned Successfully!"**
  - Additional hidden fields automatically become visible.
  - Data related to the scanned Box Number, such as **Order Number**, **Batch**, **Material**, **Material Description**, **Quantity**, and **Production Plant**, is fetched and displayed in the relevant fields and a table below.

    <img src="/DOCS_IMAGE/hht-inward/image3.jpeg" alt="Valid Barcode Details" width="250px"/>

### 2.4 Invalid Scans
- **Step 2.4.1:** If the scanned Box Number is incorrect:
  - A red message appears: **"Invalid Barcode Scan. Not Found in the FG Label Printing..!"**

    <img src="/DOCS_IMAGE/hht-inward/image4.jpeg" alt="Invalid Barcode Scan. Not Found in the FG Label Printing..!" width="250px"/>

### 2.5 Duplicate Scans
- **Step 2.5.1:** If the same Box Number is scanned more than once:
  - The system displays the error message: **"Duplicate - Serial Number already Scanned."**

    <img src="/DOCS_IMAGE/hht-inward/image8.jpeg" alt="Duplicate Barcode Message" width="250px"/>

---

## 3. Removing Scanned Serial Numbers

### **Step 3.1:** To remove a scanned Serial Number from the list:
  - **Step 3.1.1:** Check the checkbox next to the Serial Number.
  - **Step 3.1.2:** Scan the Serial Number again to remove it.
### **Step 3.2:** The system displays a message in blue: **"Serial number is removed."**

   <img src="/DOCS_IMAGE/hht-inward/image6.jpeg" alt="Serial Number Removed Message" width="250px"/>

---

## 4. Completing the Material Inward Process

### **Step 4.1:** After verifying the scanned data, click the **Inward** button.

  <img src="/DOCS_IMAGE/hht-inward/image5.jpeg" alt="Inward Button" width="250px"/>

### **Step 4.2:** Upon successful inward process, a green success message appears:
  - **"Successfully Material Inward Done For the Raw Material."**

    <img src="/DOCS_IMAGE/hht-inward/image7.jpeg" alt="Inward Success Message" width="250px"/>

---

This concludes the detailed process for Material Inward. If you have any questions or need further assistance, please contact the support team.
